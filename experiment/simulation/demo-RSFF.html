SR flip flop
Simulation of a SR flip-flop built using logic gates.

Start- Starts the simulation after the inputs are properly selected. Default speed is 1x Select the speed through dropdown menu

Reset - Resets the simulation

Click on the green input bits to set their values.

There are 2 bits SET and RESET. There is a clock that is manually operated. There are 2 outputs which are set as complements of each other in order to realise the circuit.

Top is output 1(Q) and the bottom is output (Q'), Initially the flip flop is in the reset state and Q=0, Q'=1.

The operations are as follows: SET: sets Q=1, Q'=0(SET=1), RESET: sets Q=0, Q'=1(RESET=1)

SET=0 and RESET=0: outputs the memory state
