D flip flop
Simulation of a D flip-flop built using an SR flip flop.

Start- Starts the simulation after the inputs are properly selected. Default speed is 1x Select the speed through dropdown menu

Reset - Resets the simulation

Click on the green input bits to set their values.

This is a D flip flop. It has a Data bit. There is a clock that is manually operated. There are 2 outputs

In order to realise the circuit , they are set as complements of each other. Top is output 1(Q) and the bottom is output (Q')

Initially the flip flop is in the reset state and Q=0, Q'=1.

The operations are as follows: SET: sets Q=1, Q'=0(DATA=1), RESET: sets Q=0, Q'=1(DATA=1)
